### クイックスタートVueCLI

npmのオプションでグローバルでインストールする

```sh
npm install -g @vue/cli
npm install -g @vue/cli-service-global

vue create helloworld
 ---> Default ([Vue 2] babel, eslint)

npm run serve
```

※　NuxtとVueCLIは別物！？
NUXT：Nuxt.jsは、Vue.jsのオールインワンのフレームワーク
VueCLI：Vue.jsのオールインワンツール

```sh
vue serve
```
